Operating system

De logische laag van een apparaat bestaat uit software die ervoor zorgt dat toepassingen op het apparaat uitgevoerd kunnen worden. Bij een smartphone bestaat deze software voornamelijk uit het besturingssysteem, ofwel het operating system. Deze regelt de communicatie tussen de toepassingen laag en de fysieke laag. Daarnaast regelt het operating system de communicatie tussen onderdelen binnen de computer.

Het besturingssysteem heeft 3 taken:

- beheer van randapparatuur: door middel van drivers alle hardware-onderdelen van een apparaat aansturen.

- bestandsbeheer: onthouden waar data is opgeslagen en kijken waar ruimte is voor het opslaan van nieuwe data.

- taakbeheer: alle taken die door een computer uitgevoerd moeten worden. De ‘aandacht’ van de CPU moet worden verdeeld over al deze taken, iets waar het operating system bij helpt.
