Interfaces

Een computer heeft interfaces nodig om te kunnen communiceren tussen de lagen en met zijn gebruiker. Er zijn verschillende soorten interfaces:

Hardware-interfaces:

Verbindingen tussen de fysieke onderdelen van een computer. In de meeste gevallen zijn deze geïntegreerd in een printplaat.

Software-interfaces:

Application Programming Interfaces (API’s) zijn verantwoordelijk voor de communicatie tussen software-onderdelen. Dit zijn dus de interfaces tussen de logische laag en de toepassingen laag. Deze zorgen ervoor dat er niet aparte versies nodig is van een app om hem op verschillende apparaten te kunnen installeren.

Drivers:

Een driver is een interface tussen de logische laag en de fysieke laag, die ervoor zorgt dat het operating system alle informatie heeft die nodig is om een hardware-onderdeel aan te kunnen sturen.

User-interfaces:

Een user-interface wordt gebruikt zodat een apparaat kan communiceren met zijn gebruiker en andersom. Meestal gebeurt dit via de Graphical User Interface. Voorbeelden zijn een touchscreen of een muis.
